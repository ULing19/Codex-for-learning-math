# 高数上册知识点总表

> 使用顺序：先看前置知识，再过核心知识点，最后拿题目做归类。

## 第1章 函数与极限
- 前置知识：初等函数、定义域、图像变换、绝对值与不等式
- 核心知识点：函数概念、数列极限、函数极限、无穷小与无穷大、等价无穷小、两个重要极限、连续与间断点、连续函数性质
- 常见题型：极限计算、连续性判断、分段函数求参数、极限存在性讨论
- 易错点：左右极限不一致、定义域遗漏、等价无穷小使用条件不清

## 第2章 导数与微分
- 前置知识：极限、切线斜率、初等函数求值、复合函数
- 核心知识点：导数定义、求导法则、高阶导数、隐函数求导、参数方程求导、相关变化率、微分与线性近似
- 常见题型：求导计算、切线方程、法线方程、相关变化率、微分近似
- 易错点：链式法则漏乘、dy 与 Δy 混淆、隐函数求导时把变量当常量

## 第3章 微分中值定理与导数应用
- 前置知识：导数、连续性、单调性、凹凸性、泰勒展开思想
- 核心知识点：罗尔定理、拉格朗日中值定理、柯西中值定理、洛必达法则、泰勒公式、单调区间、极值最值、凹凸与拐点、曲率、近似计算
- 常见题型：证明不等式、判单调凹凸、极值最值、极限与洛必达、函数图像分析
- 易错点：定理条件漏检、泰勒项次写错、极值点与拐点混淆

## 第4章 不定积分
- 前置知识：求导公式、复合函数、换元思想、分式分解
- 核心知识点：原函数与不定积分、换元积分法、分部积分法、有理函数积分、常用积分公式
- 常见题型：凑微分、换元、分部、部分分式分解、三角代换
- 易错点：忘记积分常数、换元后不回代、分部积分顺序选错

## 第5章 定积分
- 前置知识：面积与累积思想、极限、原函数、定积分定义
- 核心知识点：定积分定义与性质、牛顿-莱布尼茨公式、换元与分部、反常积分、审敛法、Gamma函数
- 常见题型：定积分计算、区间拆分、反常积分判敛、积分性质应用
- 易错点：上下限变换漏改、反常积分分段不当、符号与奇偶性处理错误

## 第6章 定积分的应用
- 前置知识：平面图形、函数图像、定积分计算、元素法
- 核心知识点：面积、体积、弧长、功、压力、引力、旋转体、微元法建模
- 常见题型：求曲线围成面积、旋转体体积、弧长、物理应用题
- 易错点：积分区域判断错、上下函数选反、建模时微元表达不规范

## 第7章 微分方程
- 前置知识：导数、积分、指数对数函数、方程求解
- 核心知识点：微分方程基本概念、可分离变量方程、齐次方程、可化为齐次、一阶线性方程、伯努利方程、可降阶方程、高阶线性方程、常系数齐次与非齐次、欧拉方程
- 常见题型：求通解、特解、初值问题、特征方程、待定系数法
- 易错点：特征根分类出错、特解形式设错、初值代入漏常数
