# Code of Conduct / 行为准则

## Our Pledge / 我们的承诺

This project is a learning resource for postgraduate entrance exam math review. We welcome Chinese and English issues, suggestions, corrections, and pull requests from learners, teachers, and maintainers.

本项目是面向考研数学一复习的学习资源。我们欢迎学习者、教师和维护者使用中文或英文提交 issue、建议、勘误和 PR。

We pledge to keep collaboration respectful, evidence-based, and focused on improving the handbook.

我们承诺保持尊重、基于证据、聚焦于改进手册本身的协作氛围。

## Expected Behavior / 期望行为

- Discuss formulas, examples, UI issues, and exam applicability with concrete evidence.
- Point out mistakes directly, but explain why they are mistakes and how to verify the fix.
- Prefer reproducible reports: formula card id, screenshot, browser, URL, or failing command.
- Respect different learning backgrounds. A question that seems basic may still reveal a real usability problem.

## Unacceptable Behavior / 不接受的行为

- Personal attacks, harassment, discriminatory language, or deliberate intimidation.
- Repeated off-topic comments that prevent maintainers from handling formula or UI work.
- Posting private information, exam leaks, paid copyrighted material, or unrelated advertising.
- Submitting large copied text from textbooks or websites without permission or attribution.

## Enforcement / 执行方式

Maintainers may edit, hide, or close comments and issues that violate this code. Repeated or severe violations may lead to temporary or permanent blocking from the repository.

维护者可以编辑、隐藏或关闭违反本准则的评论和 issue。重复或严重违反者可能会被临时或永久限制参与仓库。

## Reporting / 反馈方式

For ordinary formula, content, or UI problems, please open an issue using the templates in `.github/ISSUE_TEMPLATE/`.

For sensitive conduct concerns, contact the repository owner through GitHub profile contact channels instead of opening a public issue.
